# 再考裁

「再考裁」は日本の最高裁判所の判例情報をわかりやすく説明するためのウェブアプリケーションです。

## Features
- 最高裁判所の判例情報を収集し、わかりやすく表示しています
- 生成AIによって判例の要約や解説を提供します
- どの裁判官が過去にどのような判決を下したのか確認できます

## Inspiration
再考裁は、[チームみらい](https://team-mir.ai/)が提供する「[みらい議会](https://gikai.team-mir.ai/)」から着想を得て開発されました。
「みらい議会」では「国会」を対象にした説明を提供していましたが、再考裁では「最高裁判所」の判例情報に焦点を当てています。

複雑な情報を民間向けにわかりやすく提供する。という理念を先駆したチームみらいに敬意を表します。
