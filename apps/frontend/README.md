# Frontend

TanStack Start を使ったフロントエンドです。

## 実行
```bash
pnpm --filter frontend dev
```

## ビルド
```bash
pnpm --filter frontend build
```

## プレビュー
```bash
pnpm --filter frontend preview
```

## Lint
```bash
pnpm --filter frontend lint
```

## テスト
```bash
pnpm --filter frontend test
```

## 型チェック
```bash
pnpm --filter frontend typecheck
```

## 環境変数
- `TURSO_DATABASE_URL`
- `TURSO_AUTH_TOKEN`
